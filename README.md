Projet de création d'un morpion avec du python utilisable dans un terminal


le projet conciste à crée un Morpion dans un premier temps en affichage sur terminal python puis dans un second temps de crée une instance PyGames 

Le projet sur terminal propose 2 mode de jeux :

 - un mode de jeu avec un joueur contre joueur permettant de faire affronter deux joueurs ensemble sur la même machines
 - un mode de jeu avec un joueur contre l'ordinateur , avec le fait que l'ordinateur joue de façon automatique contre le joueur A
